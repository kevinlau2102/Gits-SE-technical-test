### Installation
```bash
npm install
```

### Run the app

```bash
npm install -g nodemon sequelize-cli
```

```bash
# Create Database
sequelize db:create

# Migrate Database
sequelize db:migrate

# Seeder *Optional
sequelize db:seed:all

# Run the app
npm run dev
```
